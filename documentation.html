Movie_API Documentation

Movie_API is the serverside component of a web application named myFlix. The web application will provide users with access to information about different movies, directors, and genres.

Data Formats for HTTP Requests/Responses
Description Method URL Request Parameter Body of Request Body of Response
Return a list of ALL movies to the user GET /movies None None A JSON object holding data about all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user GET /movies/[Title] /movies/:Title None A JSON object holding data about a single movie. Example:
{ "Description": "Two imprisoned men bond over a number of years, finding solace and eventual redemption through acts of common decency.",
"Genre": " Drama",
"Director": "Frank Darabont",
"image_URL": "https://m.media-amazon.com/images/M/MV5BNjQ2NDA3MDcxMF5BMl5BanBnXkFtZTgwMjE5NTU0NzE@._V1_CR0,60,640,360_AL_UX477_CR0,0,477,268_AL_.jpg"
}
Add data for new movies POST /movies None A JSON object adding new movies data. Example: {title, director, genere} A JSON object holding data about the movie that was added.
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) DELETE /movies/:id /movies/:id None A JSON object holding data about the updated list.
