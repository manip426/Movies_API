Movie_API Documentation

Movie_API is the serverside component of a web application named myFlix. The web application will provide users with access to information about different movies, directors, and genres.

Data Formats for HTTP Requests/Responses
Description Method URL Request Parameter Body of Request Body of Response

{ Genre: {

}

{ FavoriteMovies: [12-byte BSON type],
